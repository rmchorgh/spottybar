# SpottyBar

A CLI Tool for SpotifyD written in Rust.

## Usage

| Action        | Verb                                                                     |
| ------------- | ------------------------------------------------------------------------ |
| Current Track | `spottybar curr` or `spottybar current` or `spottybar currently-playing` |
| Next          | `spottybar next`                                                         |
| Previous      | `spottybar prev` or `spottybar previous`                                 |
| Play          | `spottybar play`                                                         |
| Pause         | `spottybar pause`                                                        |
